How to copy lines / characters in VI

VI is a very powerful tool to edit documents in *nix based systems. In fact, I am writing this page on VI only. However, people have been kind of spoilt by the point and click and drag and right click world of the GUI based systems and do not really make use of the powerful features and tools provided by VI. If you are a Linux or Unix system administrator, you know how valuable tool VI is when you edit the configuration files.

One of the most frequent things I as a system administrator, needs to do is to copy characters / lines.

To copy characters, count the number of characters that you need to copy. You can easily find that by looking at the bottom right corner which gives the coordinates. The actual key sequence if you want to copy 3 characters is: 

3yl

Once the characters are copied, move your cursor to the location where you want to paste and type: 
p
The same technique is used to copy lines. Instead of typing 
3yl
you should type 
3yy
to copy 3 lines and then move to the line after which you want to paste these lines and type p.

Similarly to move a set of lines you can type: 

3dd
followed by 
p
at the desired location to move the lines. The same technique can be used to delete the lines.
