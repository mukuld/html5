Hello World from Python on the web

I have been a long time fan of Python language. However I have never been able to focus on it to learn completely. I will continue to try and learn as much as I can. One day, I had a thought about executing python on website. I knew it could be done as my full website runs on Zope/Plone which is written almost entirely in Python. But the question was how do I do it.

It is recommended to enable the flag for a particular directory to prevent execution on unintended scripts / programs from your web server.

I searched on the internet and promptly found this tutorial. Essentially to enable Apache web server to parse and execute python, Apache needs to be told about the dynamic content (Details are here by enabling the+ExecCGIoption in the server for the directory in which you plan to host the python, scripts.

Below here is the very first program that you will write in any of the programming language... the ubiquitous "Hello World".

#!/usr/bin/env python
# -*- coding: UTF-8 -*-

# enable debugging
import cgitb
cgitb.enable()

print "Content-Type: text/plain;charset=utf-8"
print

print "Hello World!"
