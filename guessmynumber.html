World's best Guess my Number game

This is a game for building your:

  • Guessing ability
  • Analytical ability
  • Persistence

All these qualities will help you in real world where it will be imperitive to be all three. The game starts with asking you (the player) with your guess on the number the computer has chosen. For now, the computer does not restrict the number of tries that the user can take to guess. But keep watching this space.
